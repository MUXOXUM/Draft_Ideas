Интерактивная 3D-визуализация клеточного автомата на поверхности тора.

### Основные элементы

**Визуальная модель**
Трёхмерный тор, состоящий из сетки ячеек (72×40 сегментов), каждая из которых отображается в виде четырёхугольной грани.

**Клеточный автомат**
Реализована вариация игры «Жизнь» на нестандартной топологии — ячейки, расположенные на поверхности тора, имеют 8 соседей с учётом циклических границ по обоим измерениям.

**Управление**
- Вращение камеры — перетаскивание (реализовано через arcball-механику с кватернионами)
- Панорамирование — перетаскивание правой или средней кнопкой мыши
- Масштабирование — колесо мыши
- Сброс вида — двойной клик
- Автоматическое вращение — непрерывное по трём осям, отключается при ручном взаимодействии


### Технические особенности

- 3D-проекция с динамическим расчётом глубины и сортировкой граней для корректного отображения прозрачности
- Освещение на основе нормалей к поверхности
- Адаптивная геометрия под размер экрана
- Сглаженная интерполяция поворотов камеры
