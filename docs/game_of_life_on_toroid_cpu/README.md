# Game of Life on Toroid CPU

Визуализация игры «Жизнь» Конвея на 3D-торе с интерактивной камерой.

## Как выглядит сайт

Тёмный зелёный фон, светящийся тор из ячеек. Живые клетки — ярко-зелёные, мёртвые — приглушённо-серые. Автоматическое вращение, эволюция поколений.

## Управление

- **Левая кнопка** — вращение
- **Правая / средняя кнопка** — панорамирование
- **Колесо** — масштаб
- **Двойной клик** — сброс камеры

## Интересные решения

- Тор без границ — циклические соседи через предвычисленные таблицы
- Arcball на кватернионах с плавной интерполяцией
- Z-сортировка ячеек для корректного перекрытия
