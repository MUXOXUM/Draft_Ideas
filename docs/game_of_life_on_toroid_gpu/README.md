# GoLoT GPU

Визуализация игры «Жизнь» Конвея на 3D-торе с интерактивной камерой.

## Как выглядит

Тёмный зелёный фон, тор из ~1.4 млн ячеек с процедурной сеткой. Живые клетки светятся ярко-зелёным, мёртвые — приглушённо-серыми.

## Управление

- **Левая кнопка** — вращение
- **Правая / средняя кнопка** — панорамирование
- **Колесо** — масштаб
- **Двойной клик** — сброс камеры

## Интересные решения

- **Полностью GPU-рендер** — 2 млн вершин, 6 млн треугольников
- **Правила «Жизни» в пиксельном шейдере** — 8 соседей, текстура состояния 2000×700
- **Детализированное освещение** — diffuse + rim + specular с динамической подсветкой живых клеток
- **Сглаженная сетка** — границы ячеек рисуются через UV-координаты в шейдере без дополнительной геометрии
