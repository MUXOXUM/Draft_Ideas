# Strange Attractors

Интерактивная 3D-визуализация трёх хаотических систем: аттрактора Лоренца, аттрактора Рёсслера и модифицированной системы Лу-Чэня. Проект показывает, как из простых дифференциальных уравнений возникают сложные устойчивые формы в фазовом пространстве.

## Как выглядит сайт

После открытия пользователь видит полноэкранную тёмную сцену с облаком светящихся частиц, которые непрерывно рисуют траекторию выбранного аттрактора. В левом нижнем углу отображается название текущей системы и подсказка по управлению, а в режиме камеры появляются дополнительные инструкции и экранные кнопки для desktop и mobile.

## Управление

- `ArrowLeft` / `ArrowRight`, `A` / `D`: переключение между аттракторами
- `C`: включить или выключить режим камеры
- `LMB`: вращение сцены в режиме камеры
- `RMB` / `MMB`: панорамирование в режиме камеры
- Колесо мыши: масштабирование в режиме камеры
- Двойной клик `LMB`: сброс камеры
- `Enter`: вывести текущую матрицу камеры и параметры вида в консоль
- Экранные кнопки: переключение аттрактора, включение камеры, сброс и zoom
- На touch-экранах: в режиме камеры drag работает в выбранном режиме `Rotate` или `Pan`, который переключается отдельной кнопкой

## Интересные решения

- Для численного интегрирования используется классический четырёхшаговый метод Рунге-Кутты, чтобы траектории оставались стабильными и гладкими.
- Рендер построен на `Canvas 2D` с сортировкой по глубине и мягкими световыми следами, что создаёт ощущение объёма без WebGL.
- Для каждого аттрактора заданы собственные пресеты камеры и масштаба, поэтому сцена сразу открывается в удачном ракурсе.
