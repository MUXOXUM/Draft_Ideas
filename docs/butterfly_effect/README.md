# Butterfly Effect

Визуализация эффекта бабочки — две системы Лоренца с почти одинаковыми начальными условиями.

## Как выглядит

Тёмный фон с едва заметной сеткой, две цветные траектории (голубая и коралловая), которые начинаются практически синхронно, но постепенно расходятся. В конце каждой линии — светящаяся точка.

## Управление

- **Левая кнопка** — вращение камеры
- **Правая / средняя кнопка** — панорамирование
- **Колесо** — масштаб
- **Двойной клик** — сброс камеры

## Интересные решения

- **Адаптивный рендер** — траектории динамически обрезаются до 2200 точек, сохраняя производительность
- **Ручной ввод** — можно задать разницу в начальных значениях от 0 до 10%
