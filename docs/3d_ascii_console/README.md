# 3D ASCII Console

Трёхмерный ASCII-рендерер в стиле ретро-консоли. Вращайте многогранники, ленту Мёбиуса и тор из символов прямо в терминале.

## Как выглядит

Чёрный фон, зелёный текст. Фигуры отрисовываются псевдографикой — градациями символов от пробела до `@`. Сверху может накладываться окно настроек с ползунками.

## Управление

### Консоль (`help`)
- **`render3d`** — открыть выбор фигур
- **`color green #000000`** — сменить цвета
- **`sysinfo`** — информация о системе
- **`clear`**, **`reboot`**

### 3D-сцена
- **Стрелки** — навигация по меню/настройкам
- **M** — открыть настройки фигуры
- **ESC** — назад
- **LMB** — вращение (Arcball)
- **RMB / MMB** — панорамирование
- **Колесо** — масштаб
- **Двойной клик** — сброс камеры

## Интересные решения

- **ASCII-рендеринг** — объёмные фигуры из символов с z-буфером и освещением
- **Персистентность** — настройки и история команд сохраняются в localStorage
- **Arcball на кватернионах** — плавное вращение в ASCII
