# Optical Illusions

Минималистичная галерея оптических иллюзий на чистых HTML, CSS и JS. Проект показывает подборку изображений без лишнего интерфейса, чтобы внимание оставалось только на визуальном эффекте.

## Как выглядит сайт

После открытия пользователь видит одну иллюзию в центре экрана на темном фоне. На desktop поверх изображения нет навязчивых панелей, а на mobile появляются компактные кнопки для переключения между кадрами.

## Управление

- `←` и `→` для переключения между изображениями
- `A` и `D` для переключения между изображениями
- `Ф` и `В` для русской раскладки
- Кнопки `Назад` и `Дальше` на touch-экранах
- Свайп влево или вправо на мобильных устройствах

## Интересные решения

- Интерфейс намеренно сведен к минимуму, чтобы не спорить с самими иллюзиями
- Галерея зациклена, поэтому переходы работают без тупиковых состояний
- Изображения подгружаются по соседству с текущим кадром, а не все сразу, чтобы стартовая загрузка оставалась легче
