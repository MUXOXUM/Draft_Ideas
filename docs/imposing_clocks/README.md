# Imposing Clocks

Мини-проект с двумя альтернативными интерфейсами отображения времени. Он показывает, как одна и та же функция часов может восприниматься по-разному через классический 24-часовой циферблат и через более экспериментальную ротационную композицию.

## Как выглядит сайт

На экране показывается один из двух полноэкранных вариантов часов с тёмной атмосферной подачей. Первый экран выглядит как 24-часовой циферблат с визуальным разделением дня и ночи по данным о восходе и закате, а второй представляет время в виде трёх концентрических вращающихся колец для часов, минут и секунд.

## Управление

- `ArrowLeft` / `ArrowRight` переключают варианты часов.
- `A` / `D` тоже переключают варианты часов.
- Русская раскладка `Ф` / `В` поддерживается для той же навигации.
- Кнопки `Назад` и `Вперед` внизу экрана позволяют переключаться на mobile и desktop без клавиатуры.

## Интересные решения

- Для первого режима используется внешний API восхода и заката, но при ошибке сети проект переходит на fallback со стандартными значениями дня и ночи.
- Один проект объединяет сразу два самостоятельных способа чтения времени без переходов на другие страницы.
