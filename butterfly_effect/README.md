# Butterfly Effect

**Butterfly Effect** — это интерактивная веб-страница с двумя визуализациями аттрактора Лоренца, которые стартуют почти из одинаковых начальных условий.

Пользователь задаёт отличие в начальных значениях в процентах, например `0.001 %`, и сразу видит, как две почти идентичные траектории со временем расходятся. Проект иллюстрирует чувствительность хаотических систем к малым изменениям исходных параметров.
