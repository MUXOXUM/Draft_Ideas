### Идея проекта

Интерактивная 3D-визуализация систем хаотической динамики — странных аттракторов Лоренца, Рёсслера и модифицированной системы Лу-Чэня.

### Функциональность

Проект отображает движение тысяч частиц, подчиняющихся дифференциальным уравнениям нелинейной динамики. Каждая точка трассирует траекторию в фазовом пространстве, создавая характерные структуры аттракторов.

**Базовый режим**
- Переключение между тремя аттракторами клавишами ← → или A/D
- Название текущей системы отображается в левом нижнем углу

**Режим камеры (C)**
- Вращение сцены (LMB) через аркболь-контроллер
- Панорамирование (RMB/MMB)
- Масштабирование колесом мыши
- Сброс вида двойным кликом
- Вывод текущей матрицы поворота и параметров вида в консоль (Enter)

### Техническая реализация

- Четырёхшаговый метод Рунге-Кутты для численного интегрирования
- Рендеринг с учётом глубины и перспективы
- Адаптивная масштабируемость под размер окна
- Поддержка аппаратного ускорения через Canvas 2D
