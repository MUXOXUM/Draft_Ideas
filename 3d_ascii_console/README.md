# ASCII_OS Console

Интерактивный ASCII-эмулятор выдуманной системы `ASCII_OS` с терминальным вводом и рендерингом 3D-фигур прямо в текстовой консоли.

## Что есть сейчас

- стартовый экран только с приглашением `root@host ~>`;
- терминальные команды `help` и `render3d`;
- подсказка `type help` при неизвестной команде;
- отдельное консольное меню выбора фигур после запуска `render3d`;
- фигуры `torus`, `cube`, `octahedron`;
- рендер ASCII-символами внутри псевдотерминала;
- скрытое центральное ASCII-меню настроек, которое открывается по `M`;
- управление камерой:
  - `LMB` с зажатой кнопкой: arcball-вращение;
  - `RMB` или `MMB`: панорамирование;
  - колесо мыши: зум;
  - двойной `LMB`: сброс вида;
- ASCII-ползунки в окне настроек:
  - `size`;
  - `speed`;
- автоматическое вращение фигуры после запуска;
- временная пауза автоворота при ручном взаимодействии и автоматическое возобновление спустя короткое время.

## Управление с клавиатуры

- в терминале можно печатать команды напрямую;
- `help`: показывает список команд;
- `render3d`: открывает меню выбора фигуры;
- подсказки по управлению фигурой выводятся внутри меню `render3d`;
- `↑` / `↓`: переход по меню фигур и по ползункам;
- `Enter`: запуск выбранной фигуры;
- `M`: открыть или скрыть меню настроек во время рендера;
- `←` / `→`: изменение значения активного ползунка;
- `Esc`: закрытие меню настроек, а затем возврат в терминал.

## Идея для развития

Структура проекта сделана так, чтобы потом можно было добавлять новые фигуры и дополнительные ASCII-параметры без переделки интерфейса.
