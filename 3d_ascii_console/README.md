Интерактивный ASCII-эмулятор выдуманной системы `ASCII_OS` с терминальным вводом и рендерингом 3D-фигур прямо в текстовой консоли.

### Основные компоненты

1. **Эмуляция командной строки**
   * Пользователь взаимодействует через псевдо-терминал с приглашением в формате `[браузер]@[ОС] ~>`
   * Поддерживаются базовые команды: `help`, `pwd`, `sysinfo`, `clear`, `color`, `reboot`

2. **3D-рендер в ASCII-графике**
   * Трёхмерные объекты (сфера, тор, куб, октаэдр, икосаэдр, звездчатый многогранник, лента Мёбиуса) визуализируются символами псевдографики
   * Используется собственная система растеризации треугольников без WebGL — только вычисления на CPU
   * Динамическое освещение с настраиваемым источником света (азимут, угол места, интенсивность)

3. **Интерактивное управление 3D-сценой**
   * Вращение камеры (arcball), панорамирование и масштабирование через мышь/тачпад
   * Автоматическое вращение объекта с возобновлением после взаимодействия
   * Настройка параметров отображения (размер, скорость вращения, разрешение, отображение обратных граней)

4. **Система сохранения состояния**
   * Вся конфигурация (цвета терминала, история команд, настройки фигур, текущий режим) сохраняется в `localStorage`
   * При перезагрузке интерфейс восстанавливается в том же состоянии

### Управление с клавиатуры

В терминале можно печатать команды напрямую:
- `help`: показывает список команд;
- `render3d`: открывает меню выбора фигуры;
- `sysinfo`: показывает доступную браузеру информацию о системе;
- `clear`: очищает текст терминала;
- `color`: меняет цвет текста и фона консоли;
- `reboot`: очищает сохранённые данные `ASCII_OS` из памяти браузера;

Подсказки по управлению фигурой выводятся внутри меню `render3d`:
- `↑` / `↓`: переход по меню фигур и по ползункам;
- `Enter`: запуск выбранной фигуры;
- `M`: открыть или скрыть меню настроек во время рендера;
- `←` / `→`: изменение значения активного ползунка;
- `Esc`: закрытие меню настроек, а затем возврат в меню выбора фигуры.
