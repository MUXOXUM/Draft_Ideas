Сайт представляет собой интерактивную 3D-визуализацию клеточного автомата «Игра Жизнь», развернутую на поверхности тора.

## Основные компоненты

**Визуализация**
- Трехмерная модель тора, построенная из множества сегментов (2000×700 ячеек)
- Динамическое освещение с расчетом бликов, диффузного света и свечения по краям
- Визуальное различие между живыми (зеленые оттенки) и мертвыми (темно-серые) клетками

**Симуляция**
- Реализация классических правил Conway's Game of Life на поверхности тора
- Периодические граничные условия (тороидальная топология)
- Случайная инициализация начальной конфигурации

**Управление**
- Вращение камеры с помощью перетаскивания (arcball-интерфейс)
- Панорамирование (правая кнопка мыши)
- Масштабирование колесиком мыши
- Двойной клик для сброса вида
- Автоматическое вращение модели

**Техническая реализация**
- WebGL для аппаратно-ускоренной графики
- Два прохода рендеринга: симуляция на текстурах и визуализация геометрии
