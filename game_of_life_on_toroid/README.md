# GoL Torus

**GoL Torus** — это веб-приложение, которое моделирует классическую "Игру жизни" на тороидальной сетке и визуализирует её в виде трёхмерного вращающегося тора.

### Симуляция Game of Life

* Классические правила:

  * Рождение: 3 соседа
  * Выживание: 2 или 3 соседа
* Торроидальная топология:

  * края сетки "замкнуты"
* Автоматическое обновление поколений
