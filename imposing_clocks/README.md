Веб-страница представляет собой галерею из двух альтернативных интерфейсов отображения времени.

- **Первый интерфейс** — классические 24-часовые часы с циферблатом. Особенность заключается в визуальном разделении дня и ночи с помощью данных о времени восхода и заката солнца (получаемых через внешний API). Сектора подсвечиваются, а стрелки отображают текущее время в 24-часовом формате.

- **Второй интерфейс** — ротационный циферблат, где цифровые шкалы часов, минут и секунд размещены на концентрических кольцах. Каждое кольцо вращается независимо, а неподвижный указатель служит точкой отсчета.

Навигация между двумя вариантами осуществляется с помощью клавиатуры (стрелки влево/вправо или клавиши A/D).
