# Mystery.zip

Небольшой интерактивный проект про зашифрованный ZIP-архив, который является почти неразрешимой загадкой для обычного перебора.

## Как выглядит сайт

После открытия пользователь видит крупную стилизованную папку-архив в центре экрана. После первого клика раскрываются две боковые панели: слева объясняется идея архива и его стойкости, справа находится рабочая зона для загрузки, проверки пароля и запуска brute force.

## Управление

- Мышь: клик по папке открывает интерфейс с текстовыми экранами.
- Мышь: клик по кнопке `Скачать архив` скачивает `mystery.zip`.
- Мышь: клик по кнопке `Проверить` вручную проверяет пароль для загруженного архива.
- Мышь: клик по кнопке `Запустить brute force` запускает или останавливает автоматический перебор.
- Колесо и жесты: отдельного взаимодействия не предусмотрено.

## Интересные решения

- Для перебора используется `Web Worker`, поэтому интерфейс остаётся отзывчивым даже во время фоновых вычислений.
- Состояние архива, найденного пароля, прогресса перебора и факт открытия экранов сохраняются в `localStorage`, что позволяет восстанавливать сессию после перезагрузки.
